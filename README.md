## To run app locally

To run the app locally, follow the below steps:
1. Clone the app

`git clone https://github.com/jahnavi0324/travel-website.git`

2. Go to the folder

`cd travel-website`

3. Install the dependencies

`npm install`

4. Run the app

`npm start`
